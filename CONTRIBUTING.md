# Contributing to AetherSelf Web

## Quick Start

```bash
# Clone
git clone https://github.com/10xdev4u-alt/aetherself-web
cd aetherself-web

# Serve locally
python3 -m http.server 8080
# or
npx serve .
```

## Guidelines

- Keep it simple (no build tools)
- Vanilla HTML/CSS/JS only
- Mobile-first responsive design
