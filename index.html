FAQ

Is AetherSelf free?

Yes! AetherSelf is open source and free to use. MIT licensed.

Do I need a blockchain?

No! AetherSelf uses simple Ed25519 keys for identity. No blockchain required.

Can I sync across devices?

Yes! AetherSelf supports optional sync via WebSocket and Yjs CRDT.

Is my data really encrypted?

Yes! All data is encrypted client-side with XChaCha20-Poly1305. The server never sees plaintext.

What AI apps work with AetherSelf?

Any app that integrates the AetherSelf client SDK. We're building integrations for popular AI tools.

How do I get started?

Install the CLI, run 'aetherself init', and start the daemon. See our Quick Start guide.
